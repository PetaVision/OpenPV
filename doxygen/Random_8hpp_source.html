7  * A class to manage Tausworth random number generators so that
8  * random number generation is not affected by details of the
9  * MPI configuration.
10  *
11  * Random(HyPerCol * hc, const PVLayerLoc * locptr, bool isExtended)
12  * creates an array of RNG states.
13  * The size of the array is determined by the local layer size specified in locptr,
14  * but the initial seeds are determined by global index.
15  * Hence the sequence of random numbers generated by a specific RNG is determined
16  * by global index, not local index, so it is independent of MPI configuration.
17  */
