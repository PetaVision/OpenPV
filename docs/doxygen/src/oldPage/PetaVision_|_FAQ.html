How do I build PetaVision? How do I run it and will it run in parallel?

We use CMAKE through the command line to build Petavision. To run, make the CMAKE Makefile and go the the debug folder and run from there. Multiple core runs can be done through "make -j", where j is the number of cores used. This can all be found by going to the sidebar and to Documentation - Installations - CMAKE.

What is the params file?

The params file is a text file that contains the instructions (or parameters) that petavision follows when it gets built.

What sort of applications can I build with PetaVision? Are there examples?

Image and video recognition is currently the main application of Petavision.

How can I contribute? What is a sandbox?

Contact one of the devs if you want to be a developer or use a sandbox. Sandboxes are pieces of code that are available to everyone and can be compiled seperately from the main part of Petavision.
